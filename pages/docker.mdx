## Running app with docker

#### _Clone the code from repo locally from GitHub_

```text copy
git clone https://github.com/mairh/event-booking.git
```

#### _Nativate to folder_

```text copy
cd event-booking
```

#### _Check Node via nvm_

```text copy
nvm use
```

#### _Install dependencies_

```text copy
yarn
```

#### _Run project with docker_

```text copy
docker compose up -d
```

Then open http://localhost:3000 in your browser for event booking app

Then open http://localhost:4000 in your browser for admin panel
